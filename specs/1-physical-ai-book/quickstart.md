# Quickstart Guide: Physical AI & Humanoid Robotics Book

## Prerequisites

Before you begin working with the Physical AI & Humanoid Robotics Book content, ensure you have the following:

### System Requirements
- Node.js 18+ installed
- Git version control system
- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Terminal/command line access

### Optional Development Setup
- ROS 2 (Humble Hawksbill or later) for hands-on exercises
- NVIDIA Isaac Sim for simulation examples
- Gazebo Garden for alternative simulation
- Python 3.8+ for ROS 2 interactions

## Getting Started

### 1. Access the Book
The book is available as web-based documentation at the deployed URL. For local development:

```bash
git clone [repository-url]
cd [repository-name]
npm install
npm start
```

### 2. Create an Account
1. Navigate to the book website
2. Click "Sign Up" to create your account
3. Provide your background information:
   - Software experience level
   - Hardware experience level
   - Domain of interest
4. Select your user type: Student, Educator, or Practitioner

### 3. Navigate the Content
The book is organized into 4 main modules:

1. **Module 1: The Robotic Nervous System (ROS 2)** - Learn about ROS 2 as middleware for robotic control
2. **Module 2: The Digital Twin (Gazebo & Unity)** - Understand physics simulation in Physical AI
3. **Module 3: The AI-Robot Brain (NVIDIA Isaac)** - Explore Isaac Sim and AI control systems
4. **Module 4: Vision-Language-Action (VLA)** - Discover voice-to-action and cognitive planning

### 4. Learning Path Options
- **Complete Book**: Progress through all modules sequentially
- **Module-Specific**: Focus on specific modules relevant to your interests
- **Capstone Project**: Integrate all concepts in the autonomous humanoid project
